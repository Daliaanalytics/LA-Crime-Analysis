
#  Los Angeles Crime Analysis — Python + SQL + Power BI

## Project Overview
Analysis of 752,902 crime records in Los Angeles from 2020 to present,
identifying crime patterns, high-risk areas, and trends using Python,
SQL Server, and Power BI.

## Tools Used
-  Python (Pandas, Matplotlib) — Data Cleaning & EDA
-  SQL Server (SSMS) — Database & Queries
-  Power BI — Dashboard

## Dataset
- **Source**: [LA Crime Data - Kaggle](https://www.kaggle.com)
- **Size**: 752,902 records | 21 columns
- **Period**: 2020 — Present

## SQL Analysis
- Top 10 Crime Types
- Top 10 High-Crime Areas
- Most Dangerous Hours
- Crimes by Year (2020-2023)
- Case Resolution Status
- Top 10 Weapons Used
- Crimes by Age Group

## Dashboard Pages
- **Page 1 — Overview:** Total Crimes, Top Crime Types, Top Areas, Cases by Status, Crimes by Year
- **Page 2 — Analysis:** Crimes by Age Group, Crimes by Hour, Top 10 Weapons

## Key Findings
- **Vehicle Stolen** is the most common crime with **80,437** cases
- **Central** is the highest crime area with **50,499** cases
- **12 PM** is the most dangerous hour with **51,385** crimes
- **2022** had the highest crime count with **233,655** cases
- **79%** of cases are still under investigation
- **Strong-Arm** is the most used weapon with **140,485** cases
- **Adults (31-45)** are the most targeted age group
